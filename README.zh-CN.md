# 介绍
**让浏览器扩展的消息传递变得简单。功能完备。** 减少跨扩展各部分保持数据同步的麻烦和复杂工作。`webext-bridge` 是一个微小的库，提供了简单一致的 API，用于在浏览器扩展的不同部分（如 `background`、`content-script`、`devtools`、`popup`、`options` 和 `window` 上下文）之间发送和接收消息。
